# Расширенный двусвязный список

Реализация **расширенного двусвязного списка** на Python. Проект включает базовый класс `LinkedList` и класс-наследник `ExtendedLinkedList` с **дополнительными методами** для сложных операций.

## **Основные возможности**

- **Добавление и удаление** элементов по индексу и значению
- **Поиск элементов** с начала или конца списка
- **Печать списка** в прямом и обратном порядке
- **Получение длины** списка

## **Структура проекта**

- `double_linked_list.py` - **базовый класс** двусвязного списка
- `linked_list.py` - **расширенный класс** с выполнением задания
- `requirements.txt` - зависимости проекта

Для запуска демонстрации выполните: `python linked_list.py`
